nl.han.ica.plugins.plugin

Namespace for functions regarding the loading and evaluating of javascript plugins
from the cache or repository.

assess

(assess plugin-name correct-answer given-answer)
Executes the assess function from the javascript plugin with the parameters:
plugin-name: the class name of the plugin to access assess function.
correct-answer: answer to check against
given-answer: answer given by student.

cache-path

Path where javascript files should be cached.

file

get-plugin
(get-plugin name)
If the javascript plugin is not in the cache, load it from the repository, store it in the cache and
load it from there. Else load it from the cache.
load-from-cache
(load-from-cache path)
Loads a javascript plugin file from cache.

load-from-repo
(load-from-repo name)
Load a javscript plugin file from a repository.

load-plugin
(load-plugin name)
Load a javascript plugin

repository-address
Repository address of the plugin repository.

request-javascript-from-file
(request-javascript-from-file name)
Gets javascript from the plugin repository of a plugin.

write-to-file
(write-to-file name javascript)
Writes the javascript which is retrieved from the repository to the cache in a javascript file
with a matching name.
